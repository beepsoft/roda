Help

RODA enables preservation experts to express themselves via the functionality provided by the software. RODA has been designed to be easy-to-use and clear to non-technical people (e.g. end-users that are not archival or tech savvy).

Even though these concerns have always been in the mind of the developers, sometimes there is a need to further describe or explain how certain tasks are expected to be done in the software.

This is what this page is for. Here you will find useful information about the most common tasks a user can do in the repository software. The help page is organized by sections, each of each focusing on a special function of the OAIS (Open Archival Information System).
